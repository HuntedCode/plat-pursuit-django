Badge journey — the ladder, from scratch

The connective spine that threads the stages into one set. Real {{ series_title }} stages.

Stages complete in any order — so this is fabricated scattered: 1, 3, 4 done; 2, 5, 6 not. That kills the "climb / fill-to-progress" idea (a spine can't fill top-to-bottom when completion is out of order) and the single "you are here" pointer. Each node just carries its own done / to-do state; overall progress is an aggregate {{ done_count }} of {{ total_count }} count up top. The real choice below is how rich each node is: a dot, a number, or the stage art.
