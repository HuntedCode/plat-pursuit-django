5b. Set engraving + current-cycle print

Two new pieces, one metaphor: etched chrome stays permanent (mint engraving + set number, both deboss-shadowed into the chrome surface), ink gets reprinted each maintenance cycle (the name, dates, rarity, and the new "current cycle" line). Left card: pristine post-earn, no maintenance has happened yet so no current-cycle print. Right card: post-repair, the print has been refreshed with the latest cycle's standing. The set number in the bottom-right is shared across every badge of the same series + tier — like a print-run stamp on a card set.

5c. Maintenance state

A maintenance card is one that's been earned before (mint and set engravings permanent) but is currently dormant — a new game in the series has shipped and the user needs to re-earn the badge to reactivate it. The badge sleeps cool while every repair-CTA signal stays warm — the temperature contrast carries the whole mood: dormant, waiting, but unmistakably labeled for action.

5d. Active → Maintenance moment

The choreographed transition from active to maintenance. Fired only when a user is first being told their badge has entered a maintenance cycle (e.g. on notification click) — not on every render of a maintenance card. ~2.55 seconds. The card starts in its full active appearance (mint engraving, set engraving, earned date, full-color badge), then: badge cools to dormant, plinth fades, repair signals fade in, the RESTORING banner slides down from above, the MAINTENANCE band drops from above with a slight bounce and impact shake on landing, the meta line swaps from "Earned [date]" to "4 of 5 repaired," and the REACTIVATE label settles in as the wake-up call.
