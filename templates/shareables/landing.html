{% extends 'base.html' %} {% load static custom_filters %} {% block title %}My Shareables{% endblock %} {% block robots %}noindex, nofollow{% endblock %} {% comment %} My Shareables landing page. A wayfinder grid that distributes users to the dedicated sub-pages for each share image type. Each card has icon + name + tagline + example image (or gradient fallback) + CTA button. The example images live at static/images/shareables/.png. The card uses CSS background-image with two layered values: the static PNG first, then a gradient as a fallback. If the PNG file doesn't exist (or fails to load), the gradient shows instead — no broken image icon, no JS error handling needed. Drop new asset PNGs into static/images/shareables/ to replace the placeholders. The landing itself queries no per-user data — sub-pages do the heavy lifting. This keeps the page fast and means new users with no platinums yet still see a useful "here's what's available" page instead of empty states for each share type.
