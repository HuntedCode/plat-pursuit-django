{% comment %} Markdown formatting toolbar. Place directly above a textarea. Buttons are organized into four semantic groups separated by dividers: 1. Inline emphasis — B / I / U / S / Spoiler 2. Block structure — H2 / H3 / H4 / List / Quote / Table / Callout 3. Inserts — Link / Image / Controller / Trophy link 4. Meta — Help (right-aligned) Each icon button carries matching `title` (tooltip) and `aria-label` (screen reader) text. flex-wrap on the container lets the toolbar collapse to multiple rows at narrow viewports without overflowing. {% endcomment %}
