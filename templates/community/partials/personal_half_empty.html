{% comment %} Reusable empty-state for the personal (bottom) half of a Community Hub feature card. Renders for anonymous viewers OR for authenticated viewers who haven't linked a PSN profile yet, picking the right CTA based on ``viewer_is_authenticated``. The block reserves the same vertical space as a populated personal half so cards stay the same height regardless of the viewer's auth state. The icon, copy, and color are all parameterized so the card can match its own accent color (Reviews=success, Lists=primary, Challenges=secondary, Leaderboards=accent). Required context: - viewer_is_authenticated: bool — set on the page by CommunityHubView - accent_color_class: e.g. "text-success" — for the icon tint - accent_border_class: e.g. "border-success/30" — for the dashed border - prompt_text: short blurb to fill in why the user might want to sign in, e.g. "Sign in to compare your reviews to the community." {% endcomment %}
