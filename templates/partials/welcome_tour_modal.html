{% load static %} {% comment %} Welcome Tour modal: a 4-step carousel introducing PlatPursuit's hub-of-hubs navigation. Auto-shows once on first dashboard load after PSN link + sync, then only re-appears via the avatar dropdown "Replay Tour" button. Included in base.html for all authenticated+linked users so the re-trigger path always has the modal in the DOM. The show_welcome_tour context var tells JS whether to auto-open on page load. Each step has two sections: 1. A "navigation path" showing the navbar buttons with the current hub highlighted, flowing into the sub-nav strip. This directly teaches users WHERE to find things in the real UI. 2. Feature preview cards showing WHAT they'll find there. See docs/design/tutorial-system.md for design rationale. {% endcomment %}
