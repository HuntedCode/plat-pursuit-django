Manual anchor PER GAME to IGDB id

Per-Game variant of the bulk anchor action. Use this when one or more Games inside a Concept actually belong to a different IGDB version than the Concept is anchored at (for example, a Remaster's trophy list living inside the original game's Concept that should split out). Enter an IGDB id per Game; each Game is routed independently to the per-version target Concept for the id you provide, find-or-creating the target as needed.

The same vetting the bulk anchor uses runs per Game: trophy-fingerprint check against the target's existing Games, plus the identity cross-check (title similarity, platform overlap, release-year gap). Clean Games move via Game.add_concept. Flagged Games get a ConceptJoinReview entry the same way the bulk path would.

Leave any row's IGDB id blank to skip that Game in this run.
