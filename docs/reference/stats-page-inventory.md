# Stats Page - Stat Inventory

> Quick-reference of every stat displayed on `/my-stats/`. Update this doc when adding or removing stats.

## Section 1: Career Overview (FREE)

| Stat | Description |
|------|-------------|
| Total Trophies | Total earned trophies |
| Bronze / Silver / Gold / Platinum | Count per trophy type |
| Type Distribution (%) | Percentage breakdown by type + stacked bar |
| Total Unearned | Unearned trophies across all games |
| Earn Rate | % of all encountered trophies earned (earned / (earned + unearned)) |
| Total Games | Games in library |
| 100% Complete | Games at 100% completion |
| Backlog | Games not yet at 100% (total games - completes) |
| Avg Completion | Average progress % across all games |
| Completion Rate | % of games at 100% |
| Platinum Rate | % of games with platinum |
| Trophies / Game | Average trophies per game |
| Trophies / Day | Lifetime average (career start = first trophy date) |
| Plats / Month | Lifetime average |
| Career Length | Years since first trophy earned |
| PSN Level | Trophy level from PSN + progress bar to next level |
| PS Plus | PS Plus membership badge |
| Hidden Games | Count of hidden/deleted games |
| Country | User's country |

## Filters

| Toggle | Effect |
|--------|--------|
| Exclude Shovelware | Removes auto/manually flagged shovelware from game-level stats and records |
| Exclude Hidden | Removes user-hidden games from game-level stats and records |

Filters persist in localStorage. Each filter combo has its own 4-hour cache key.
